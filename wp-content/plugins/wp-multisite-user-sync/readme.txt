Sync/unsync users from one site (blog) to the other sites (blogs) in your WordPress Multisite network.

== Description ==

WP Multisite User Sync plugin can sync/unsync users from one site (blog) to the other sites (blogs) in your WordPress Multisite network.

= Features =
* Sync/Unsync individual user
* Sync/Unsync to single site
* Sync/Unsync to multiple site
* Sync/Unsync to all site

= Settings =
* User Roles: Select user roles you want to give individual sync/unsync access.

= Usage =

**Individual Sync/Unsync:** You can sync/unsync individual users from one site to the other sites. Please follow the below steps.

* Edit any user
* You will see 'WP Multisite User Sync' section
* Select destination sites you want sync/unsync
* Click on 'Sync/Unsync' button

== Installation ==

This plugin only works with WordPress Multisite, will not work in a single site, and must be network activated.

* Multi-domains
* Sub-domains
* Sub-directories

= From your WordPress network dashboard =

1. Go to 'Network Admin' -> 'Plugins' -> 'Add New'
2. Search for 'WP  Multisite User Sync'
3. Activate 'WP  Multisite User Sync' plugin from your Plugins page.

= From WordPress.org =

1. Download 'WP  Multisite User Sync'.
2. Unzip and upload the 'wp-multisite-user-sync' folder to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'WP  Multisite User Sync' plugin from your Plugins page.
